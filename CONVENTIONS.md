# Conventions

## Workflow
1. Pull the latest `main`
2. Create your branch
3. Make small changes
4. Commit often
5. Push your branch
6. Open a pull request
7. Respond to review feedback in the same PR

## Local checks before pushing
- Run `npm start`
- Confirm the required topics return useful answers
- Confirm unknown questions return the fallback response
- Keep your branch limited to this assignment

## Notes
- Keep all week 1 data local
- Do not require a live Discord bot token for week 1
- If you change the FAQ structure, update the router too
